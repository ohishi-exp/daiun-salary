## テストでの使い方

### CSV比較テスト（`test_and_deploy.sh`）
```bash
./test_and_deploy.sh              # 全比較テスト実行
./test_and_deploy.sh --skip-compare  # 比較スキップ
```

期待差分数:
- 1018/1021/1026: **0件**
- 1029/1032/1036/1037: **0件**
- 1039: **15件**（既知差分）

### 手動比較
```bash
cargo run --bin compare -- test_data/csvdata-202602-1018-1021-1026.zip test_data/拘束時間管理表_202602-1018-1021-1026.csv
```
